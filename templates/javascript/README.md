## Description

This repository was built using [Nodestrap](https://github.com/o-sca/nodestrap/tree/main)

## Installation

```
$ npm run install
```

## Running the app

```
# development
$ npm run dev

# production
$ npm run start

# build
$ npm run build

# watch
$ npm run watch

# test
$ npm run test
```
